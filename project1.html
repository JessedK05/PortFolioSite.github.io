Tales of Old:

This is a project me and a couple friends started in the spring of 2022. I cannot show a lot because it's still in heavy development. The game is a 2D game inspired on Hollow Knight and Cuphead also when it comes to the difficulty of the game. We are planning to release it on steam, but we don't when ofcourse because it's still very early on in development.

Story:

The game is about a Rat who discovers a fairytales book which had (unkown to him) magical powers. This magic in the book made it so that the rat got sucked into the book and has to overcome the fairytales.
